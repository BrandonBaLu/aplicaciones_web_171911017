Estructura básica de una pagina web

Estructura básica

¿para que sirve el (>DOCTYPE html<)?

Cuando escribimos nuestro documento HTML, lo primero que tenemos que escribir es el doctype. El doctype es la declaración de tipo de documento. En otras palabras, el doctype nos sirve para indicar que nuestro documento está escrito siguiendo la estructura determinada por un DTD concreto.

¿Para que sirve (>html<)?

El elemento html contituye la raíz de todo documento HTML bien formado y se supone que encierre a todos los demás elementos entre sus etiquetas

¿Para que sirve (>head<)?

La parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javaScript, CSS…

¿Para que sirve (>title<)?

El elemento title provee un título o nombre para el documento. Este título debe proporcionar una corta pero precisa descripción del contenido del documento o su propósito aún cuando es sacada de contexto (como, por ejemplo, cuando se muestra en los resultados de una búsqueda).

¿Para que sirve (>boddy<)?

Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.
